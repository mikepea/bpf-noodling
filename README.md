# bpf-noodling

Playground for bpftrace and other eBPF/bcc learnings

## Installation

### Ubuntu Bionic

    # via https://snapcraft.io/bpftrace
    sudo snap install bpftrace
    sudo snap connect bpftrace:system-trace
